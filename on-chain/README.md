# React dApp

## Requirements
* Node.js >= 8.11.0

## Getting started

````bash
git clone git@github.com:agusferreira/find-them.git
cd find-them/on-chain
npm install
touch .env
````

A `.env` file is required. Below the variables required
  * REACT_APP_API_ROOT=http://localhost


### Running the server

````bash
npm start
````

## Build files for production

````bash
npm run build
````
